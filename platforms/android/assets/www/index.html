Plugins Cordova

  • Globalization

    cordova plugin add cordova-plugin-globalization

  • Camera

    cordova plugin add cordova-plugin-camera

  • Barcode Scanner

    cordova plugin add cordova-plugin-barcodescanner (installe aussi cordova-plugin-compat)

  • ezAR VideoOverlay

    cordova plugin add cordova-plugin-ezar-videooverlay (ou cordova plugin add com.ezartech.ezar.videooverlay)

  • Compass Orientation

    cordova plugin add cordova-plugin-device-orientation

  • Geolocation

    cordova plugin add cordova-plugin-geolocation
